# 飞书 Todo 机器人 - 功能说明

## 🚀 新增功能

### 1. 快速模式（Quick Mode）

**适用场景：** 手机使用，希望快速创建待办，无需确认步骤

**配置方法：**

在 `.env` 文件中设置：
```env
ENABLE_QUICK_MODE=true
```

**使用效果：**

**标准模式（默认）：**
```
你：明天 3点 提交周报 p1

机器人：[返回确认卡片]
📋 待办总结
1. 提交周报
   ⏰ 2026-06-02 15:00
   🔥 高优先级

[确认写入] [取消]
```

**快速模式：**
```
你：明天 3点 提交周报 p1

机器人：✅ 已创建 1 个待办

1. 提交周报
   ⏰ 2026-06-02 15:00
   🔥 高优先级
```

**区别：**
- ✅ 跳过确认步骤，直接写入多维表格
- ✅ 减少一次点击操作
- ✅ 更适合手机快速输入
- ⚠️ 无法在写入前修改，需要确保输入准确

**推荐使用场景：**
- 📱 手机端使用
- ⚡ 快速记录灵感
- 🏃 紧急任务创建
- 📝 简单明确的待办

**不推荐场景：**
- 🤔 复杂任务需要仔细确认
- 📊 批量创建多个任务
- ⚠️ 不确定解析是否准确

---

## 📋 使用建议

### 标准模式 vs 快速模式

| 特性 | 标准模式 | 快速模式 |
|------|---------|---------|
| 确认步骤 | ✅ 有 | ❌ 无 |
| 操作次数 | 2次（发送+确认） | 1次（发送） |
| 适合场景 | 电脑端、复杂任务 | 手机端、简单任务 |
| 错误修正 | 确认前可修改 | 需要手动删除 |
| 响应速度 | 稍慢 | 更快 |

### 最佳实践

1. **手机使用开启快速模式**
   ```env
   ENABLE_QUICK_MODE=true
   ```

2. **电脑使用保持标准模式**
   ```env
   ENABLE_QUICK_MODE=false
   ```

3. **混合使用**
   - 可以根据需要随时切换
   - 修改 `.env` 后重启服务即可生效

### 2. 查询任务功能 ⭐ 新增

**适用场景：** 快速查看多维表格中的所有任务

**使用方法：**

发送以下命令之一：
```
查询
任务列表
我的任务
查询任务
```

**返回示例：**
```
📋 任务列表（5 项）

1. 提交周报
   ⏰ 2026-06-02 15:00
   🔥 高优先级

2. 整理方案
   ⏰ 2026-06-02 18:00
   💡 中优先级

3. 客户会议
   ⏰ 2026-06-03 14:00
   🔥 高优先级
```

**功能特点：**
- ✅ 显示所有未完成任务
- ✅ 包含截止时间和优先级
- ✅ 交互式卡片，每个任务带操作按钮
- ✅ 支持完成、延期、删除操作
- ✅ 最多显示 10 个任务（避免卡片过长）
- ✅ 按创建时间排序

**操作按钮：**
- **✅ 完成** - 删除该任务（标记为完成）
- **⏰ 延期** - 截止时间延后一天
- **🗑️ 删除** - 永久删除该任务

**使用示例：**
```
你：查询

机器人：[返回交互式卡片]
📋 任务列表（5 项）

1. 提交周报
   ⏰ 2026-06-02 15:00
   🔥 高优先级
   [✅ 完成] [⏰ 延期] [🗑️ 删除]

2. 整理方案
   ⏰ 2026-06-02 18:00
   💡 中优先级
   [✅ 完成] [⏰ 延期] [🗑️ 删除]
```

点击按钮后会立即执行操作并显示提示。

## 💡 使用技巧

### 1. 快速输入格式

**推荐格式：**
```
[时间] [任务名] [优先级] [备注]
```

**示例：**
```
明天 3点 提交周报 p1
下周一 客户会议 p2 备注: 准备PPT
今天 晚上8点 健身 p3
```

### 2. 批量创建

用分号 `;` 分隔多个任务：
```
明天 3点 提交周报 p1；整理方案；下周一 客户会议
```

### 3. 时间表达

支持多种时间格式：
- **相对时间**：明天、后天、下周一、下个月
- **具体时间**：2026-06-01、6月1日、15:00
- **组合表达**：明天下午3点、下周一上午

### 4. 优先级标记

- `p1` / `高` / `紧急` → 高优先级
- `p2` / `中` / `普通` → 中优先级
- `p3` / `低` → 低优先级

### 5. 添加备注

使用关键词：
```
完成项目方案 备注: 需要附上数据分析
整理文档 注意: 要更新版本号
准备会议 说明: 提前15分钟到场
```

## 🔧 配置参考

### 完整配置示例

```env
# 服务配置
PORT=3000
APP_TIMEZONE=Asia/Shanghai

# 飞书应用配置
FEISHU_APP_ID=cli_xxx
FEISHU_APP_SECRET=xxx
FEISHU_VERIFICATION_TOKEN=xxx
FEISHU_ENCRYPT_KEY=xxx

# 多维表格配置
FEISHU_BASE_TOKEN=xxx
FEISHU_BASE_TABLE_ID=xxx

# AI 配置（可选）
OPENAI_API_BASE_URL=https://api.openai.com/v1
OPENAI_API_KEY=sk-xxx
OPENAI_MODEL=gpt-4o-mini
OPENAI_VISION_MODEL=

# 功能开关
ENABLE_IMAGE_RECOGNITION=true      # 图片识别
ENABLE_TODO_OPTIMIZATION=false     # 智能优化
ENABLE_AI_PARSE=true               # AI 增强解析
ENABLE_QUICK_MODE=true             # 快速模式 ⭐ 新增
```

### 推荐配置组合

**手机端优化配置：**
```env
ENABLE_QUICK_MODE=true             # 开启快速模式
ENABLE_AI_PARSE=true               # 开启 AI 解析
ENABLE_IMAGE_RECOGNITION=true      # 支持截图识别
```

**电脑端标准配置：**
```env
ENABLE_QUICK_MODE=false            # 保持确认步骤
ENABLE_AI_PARSE=true               # 开启 AI 解析
ENABLE_TODO_OPTIMIZATION=true      # 开启智能优化
```
