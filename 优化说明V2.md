### 2. **操作历史记录** ⭐⭐⭐

#### 功能特性
- ✅ 记录每次CLI操作（成功/失败）
- ✅ 记录待办创建操作
- ✅ 支持快速重复上次操作
- ✅ 提供操作统计信息

#### 数据结构
```typescript
interface OperationRecord {
  id: string;
  userId: string;
  timestamp: number;
  type: 'cli' | 'todo' | 'query';
  intent?: IntentAnalysisResult;
  description: string;
  success: boolean;
  canRepeat: boolean;  // 是否可重复
  canUndo: boolean;    // 是否可撤销
}
```

#### 使用场景

**查看历史：**
```
用户：/历史
机器人：📝 最近操作（5项）：

1. ✅ 查看日程
   🕐 06-01 14:30 🔄
2. ✅ 搜索联系人：张三
   🕐 06-01 14:25 🔄
3. ❌ 查看待审批
   🕐 06-01 14:20 🔄
...

📊 统计：成功 4 | 失败 1
```

**重复操作：**
```
用户：/重复
机器人：🔄 正在重复执行：查看日程
机器人：📅 日程安排（3项）：...
```

## 🏗️ 架构改进

### 类型增强
```typescript
// 系统命令类型
type SystemCommand = 
  | { type: 'history'; limit?: number }
  | { type: 'repeat' }
  | { type: 'cache_stats' }
  | { type: 'cache_clear'; cacheType?: ... }
  | { type: 'help' };

// 快捷命令结果
type ShortcutResult = 
  | { kind: 'query' }
  | { kind: 'cli'; intent: IntentAnalysisResult }
  | { kind: 'system'; command: SystemCommand };
```

### 处理流程
```
用户消息
   ↓
1. 解析快捷命令 (parseShortcutCommand)
   ├─ 系统命令 → handleSystemCommand
   ├─ 查询命令 → handleQueryCommand
   └─ CLI命令  → handleCLIOperation
   ↓
2. 自然语言查询 (regex match)
   ↓
3. AI意图识别 (analyzeIntent)
   ↓
4. 待办解析 (parseTodoDrafts)
```
