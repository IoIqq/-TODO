# 飞书多维表格字段配置指南

## 📋 必需字段列表

请确保你的飞书多维表格包含以下字段（字段名称必须完全匹配）：

### 1. 待办事项
- **类型**：单行文本
- **说明**：任务的标题
- **示例**：提交周报、客户会议

### 2. 优先级
- **类型**：单选
- **选项**：
  - 🔴P0-高优
  - 🟡P1-一般
  - 🟢P2-低优
- **说明**：任务的优先级，AI会自动判断

### 3. 截止日期
- **类型**：日期
- **说明**：任务的截止时间
- **示例**：2026-06-01 15:00

### 4. 是否已完成
- **类型**：复选框
- **说明**：标记任务是否完成
- **默认值**：未勾选

### 5. 执行人
- **类型**：人员
- **说明**：任务的负责人
- **备注**：会自动设置为发送消息的用户

### 6. 备注 ⭐ 新增
- **类型**：多行文本
- **说明**：任务的详细备注信息
- **示例**：需要准备PPT、记得带数据、联系张三

### 7. AI 特办事项风险总 ⭐ 新增
- **类型**：单选
- **选项**：
  - 🔴高风险
  - 🟡中风险
  - 🟢低风险
- **说明**：AI自动评估任务风险等级
- **评估规则**：
  - 🔴高风险：时间紧迫（今天内）、依赖他人、复杂任务
  - 🟡中风险：常规任务、时间充裕
  - 🟢低风险：简单任务、无时间压力

### 8. 创建时间 ⭐ 新增
- **类型**：日期时间
- **说明**：任务创建的时间戳
- **备注**：自动生成，无需手动填写

---

## 🎯 AI 功能说明

### 风险评估
AI会根据以下因素自动评估任务风险：
- **时间紧迫度**：今天内完成的任务为高风险
- **任务复杂度**：需要多步骤、依赖他人的任务风险较高
- **任务类型**：会议、汇报等关键任务风险较高

### 备注提取
AI会自动从用户输入中提取备注信息，例如：
- 输入："明天3点开会，记得带PPT"
- 提取：备注="记得带PPT"

## 🔧 配置步骤

### 1. 打开飞书多维表格
访问你的飞书多维表格（BASE_TOKEN 对应的表格）

### 2. 添加缺失字段
如果表格中没有以下字段，请手动添加：

#### 添加"备注"字段
1. 点击表格右上角的"+"添加字段
2. 字段名称：`备注`
3. 字段类型：多行文本
4. 点击确认

#### 添加"AI 特办事项风险总"字段
1. 点击表格右上角的"+"添加字段
2. 字段名称：`AI 特办事项风险总`
3. 字段类型：单选
4. 添加选项：
   - 🔴高风险
   - 🟡中风险
   - 🟢低风险
5. 点击确认

#### 添加"创建时间"字段
1. 点击表格右上角的"+"添加字段
2. 字段名称：`创建时间`
3. 字段类型：日期时间
4. 点击确认

### 3. 验证配置
发送一条测试消息给机器人：
```
今天下午3点开会，记得带PPT
```

检查多维表格中是否正确创建了记录，并包含所有字段。

---

## ⚠️ 注意事项

1. **字段名称必须完全匹配**：包括空格、大小写、标点符号
2. **单选字段的选项必须包含表情符号**：如 🔴高风险（不是"高风险"）
3. **创建时间字段会自动填充**：无需手动设置
4. **风险评估默认为中风险**：如果AI无法判断，会默认设置为🟡中风险
