# 多维表格配置指南

## 📋 需要的字段配置

为了让飞书智能todo助手正常工作，请在多维表格中配置以下字段：

### 必需字段

| 字段名 | 字段类型 | 说明 | 示例值 |
|--------|---------|------|--------|
| 待办事项 | 文本 | 任务标题 | "提交周报" |
| 优先级 | 单选 | 任务优先级 | 🔴P0-高优 / 🟡P1-一般 / 🟢P2-低优 |
| 截止日期 | 日期时间 | 任务截止时间 | 2026-06-02 15:00 |
| 是否已完成 | 复选框 | 任务完成状态 | ☐ / ☑ |

### 推荐字段

| 字段名 | 字段类型 | 说明 | 示例值 |
|--------|---------|------|--------|
| AI 特办事项风险总 | 单选 | AI评估的风险等级 | 🔴高风险 / 🟡中风险 / 🟢低风险 |
| 备注 | 多行文本 | 任务的额外说明 | "记得带数据" |
| 创建时间 | 日期时间 | 任务创建时间 | 2026-06-01 16:30 |
| 执行人 | 人员 | 任务负责人 | @张三 |

## 🔧 字段配置步骤

### 1. 打开多维表格
访问：`https://feishu.cn/base/HEAHbiuUFaQhE7siqzic9vjvnpf`

### 2. 添加缺失的字段

#### 添加"备注"字段
1. 点击表格右上角的 **"+"** 按钮
2. 选择 **"多行文本"**
3. 字段名称：`备注`
4. 点击确认

#### 添加"AI 特办事项风险总"字段
1. 点击表格右上角的 **"+"** 按钮
2. 选择 **"单选"**
3. 字段名称：`AI 特办事项风险总`
4. 添加选项：
   - `🔴高风险`
   - `🟡中风险`
   - `🟢低风险`
5. 点击确认

#### 添加"创建时间"字段
1. 点击表格右上角的 **"+"** 按钮
2. 选择 **"日期"**
3. 字段名称：`创建时间`
4. 格式：日期时间
5. 点击确认

## ⏰ 配置提醒功能

### 方式1：字段级提醒（推荐）
1. 点击"截止日期"字段的设置图标
2. 启用 **"提醒"** 功能
3. 设置提醒时间：
   - 提前30分钟
   - 提前1小时
   - 提前1天
4. 保存设置

### 方式2：自动化规则
1. 点击表格右上角的 **"自动化"** 按钮
2. 创建新规则：
   - 触发条件：截止日期临近
   - 执行动作：发送飞书通知
3. 保存规则

## 🔍 故障排查

### 问题1：字段不存在
**错误信息：** "Create base record failed"

**解决方法：**
1. 检查多维表格中是否有对应字段
2. 确认字段名称完全一致（包括空格）
3. 确认字段类型正确

### 问题2：提醒不工作
**解决方法：**
1. 检查"截止日期"字段的提醒设置
2. 确认飞书通知权限已开启
3. 检查自动化规则是否启用

### 问题3：风险评估不显示
**解决方法：**
1. 确认已添加"AI 特办事项风险总"字段
2. 确认字段类型为"单选"
3. 确认选项值完全匹配
