# 对话历史功能配置指南

本指南帮助你配置对话历史功能，让 AI 助手拥有记忆能力。

---

## 📋 第一步：创建对话历史表

### 1. 打开飞书多维表格

访问你的多维表格：
```
https://feishu.cn/base/HEAHbiuUFaQhE7siqzic9vjvnpf
```

### 2. 创建新表

点击左下角 **"+"** 按钮，创建新表，命名为：**对话历史**

### 3. 添加字段

创建以下字段：

| 字段名 | 字段类型 | 说明 |
|--------|---------|------|
| 用户ID | 文本 | 飞书用户的 open_id |
| 消息ID | 文本 | 消息的唯一标识 |
| 角色 | 单选 | user（用户）或 assistant（AI） |
| 内容 | 多行文本 | 对话内容 |
| 时间戳 | 日期时间 | 对话时间 |

### 4. 配置单选字段

**角色** 字段的选项：
- `user` - 用户消息
- `assistant` - AI 回复

### 5. 获取表格 ID

1. 点击表格右上角的 **"..."** 菜单
2. 选择 **"复制链接"**
3. 链接格式：`https://feishu.cn/base/HEAHbiuUFaQhE7siqzic9vjvnpf?table=tblXXXXXXXXXXXX`
4. 复制 `tbl` 开头的表格 ID

---

## ⚙️ 第二步：配置环境变量

编辑 `.env` 文件，添加对话历史表 ID：

```env
# 对话历史表配置
FEISHU_CHAT_HISTORY_TABLE_ID=tblXXXXXXXXXXXX  # 替换为你的表格 ID
ENABLE_CHAT_MEMORY=true                        # 启用对话记忆
CHAT_MEMORY_LENGTH=10                          # 保留最近 10 条对话
```

---

## 🚀 第三步：重启服务

1. 双击 **"停止服务.bat"**
2. 双击 **"一键启动.bat"**

---

## ✅ 第四步：测试

### 测试对话记忆

**对话 1：**
```
你：明天 3点 提交周报 p1
AI：好的，已创建任务"提交周报"
```

**对话 2：**
```
你：把它延期到后天
AI：好的，我把"提交周报"延期到后天
```

如果 AI 能理解"它"指的是"提交周报"，说明记忆功能正常！

## 📊 功能说明

### 记忆范围
- 保留最近 10 条对话（可配置）
- 按用户分组（不同用户的对话互不影响）
- 自动清理旧对话

### 隐私保护
- 对话历史存储在你的多维表格中
- 只有你能访问
- 可以手动删除历史记录

### Token 消耗
- 每次调用会包含历史对话
- 约增加 500-1000 tokens
- 建议限制历史长度

---

## 🔧 高级配置

### 调整记忆长度

编辑 `.env`：
```env
CHAT_MEMORY_LENGTH=20  # 保留 20 条对话（更长的记忆）
```

### 禁用记忆功能

编辑 `.env`：
```env
ENABLE_CHAT_MEMORY=false  # 禁用对话记忆
```

### 清除历史记录

在飞书多维表格中：
1. 打开"对话历史"表
2. 选择要删除的记录
3. 点击删除

---

## 🔍 故障排查

### 问题 1：AI 没有记忆

**检查：**
1. 确认 `ENABLE_CHAT_MEMORY=true`
2. 确认表格 ID 正确
3. 查看服务日志

### 问题 2：记忆太短

**解决：**
增加 `CHAT_MEMORY_LENGTH` 的值

### 问题 3：Token 消耗太多

**解决：**
减少 `CHAT_MEMORY_LENGTH` 的值

---

## 💡 使用建议

### 1. 合理设置记忆长度
- 日常使用：10 条
- 复杂对话：20 条
- 简单任务：5 条

### 2. 定期清理历史
- 每月清理一次
- 删除无用对话
- 保持表格整洁

### 3. 隐私保护
- 不要在对话中包含敏感信息
- 定期检查历史记录
- 必要时清除历史
