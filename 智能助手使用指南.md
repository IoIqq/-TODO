# 飞书智能助手使用指南

## 🎉 功能概述

飞书智能助手是在原有待办助手基础上的重大升级，通过集成 lark-cli，让机器人能够执行更多飞书操作，真正成为你的全能助手！

### ✨ 核心能力

**原有功能：**
- ✅ 待办事项管理
- ✅ 自然语言解析
- ✅ AI 增强功能

**新增功能：**
- 📅 **日历管理** - 查看日程、创建会议
- 👥 **联系人搜索** - 快速找人、查看信息
- 📄 **文档操作** - 搜索、创建文档
- ✅ **审批管理** - 查看待审批、提交审批
- 🎯 **任务管理** - 飞书任务列表
- 💬 **消息发送** - 发送通知

---

## 🚀 快速开始

### 1. 配置环境变量

在 `.env` 文件中添加以下配置：

```env
# CLI 智能助手配置
LARK_CLI_PATH=D:\Feishu\cli\lark-cli.cmd
ENABLE_SMART_ASSISTANT=true
CLI_TIMEOUT=30000
CLI_CONFIRM_ACTIONS=create,update,delete,send,approve
CLI_LOG_ENABLED=true
```

### 2. 确保 lark-cli 已安装

确认 `D:\Feishu\cli\lark-cli.cmd` 路径正确，并且 lark-cli 已正确配置飞书凭据。

### 3. 启动服务

```bash
npm run dev
```

## 🔐 安全机制

### 操作分类

**🟢 安全操作（无需确认）**
- 查看日程、日历
- 搜索联系人
- 查看文档列表
- 查看审批列表
- 查看任务列表

**🟡 重要操作（需要确认）**
- 创建/修改/删除日历事件
- 创建/修改/删除文档
- 提交/审批流程
- 发送消息
- 删除任何内容

### 确认流程

1. 用户发送操作请求
2. AI 识别为重要操作
3. 机器人发送确认卡片，显示操作详情
4. 用户点击"确认执行"
5. 执行操作并返回结果

## ⚙️ 配置说明

### CLI_CONFIRM_ACTIONS

控制哪些操作需要用户确认，默认值：
```
create,update,delete,send,approve
```

可以根据需要调整，例如：
- 只确认删除操作：`delete`
- 确认所有写操作：`create,update,delete,send,approve,post`

### CLI_TIMEOUT

CLI 命令执行超时时间（毫秒），默认 30000（30秒）。

如果某些操作需要更长时间，可以增加此值：
```env
CLI_TIMEOUT=60000  # 60秒
```

### CLI_LOG_ENABLED

是否记录 CLI 执行日志，默认 true。

调试时建议开启，生产环境可以关闭：
```env
CLI_LOG_ENABLED=false
```

## 🐛 故障排查

### 问题1：CLI 命令执行失败

**可能原因：**
- lark-cli 路径不正确
- lark-cli 未配置凭据
- 权限不足

**解决方法：**
1. 检查 `LARK_CLI_PATH` 配置
2. 运行 `lark-cli doctor` 检查配置
3. 确认飞书应用权限

### 问题2：AI 无法识别意图

**可能原因：**
- `ENABLE_SMART_ASSISTANT` 未开启
- OpenAI API Key 未配置
- 用户输入不够明确

**解决方法：**
1. 确认配置项已启用
2. 检查 AI 配置
3. 使用更明确的表达

### 问题3：操作需要确认但没有弹出卡片

**可能原因：**
- 卡片发送失败
- 网络问题

**解决方法：**
1. 查看服务日志
2. 检查飞书回调配置
3. 重试操作

## 📊 使用技巧

### 1. 组合使用

可以在一句话中包含多个操作：
```
帮我查一下明天的会议，然后给参会人发个提醒
```

AI 会：
1. 先查询明天的会议（安全操作，直接执行）
2. 然后询问是否发送提醒（需要确认）

### 2. 上下文理解

机器人会记住对话上下文：
```
用户：明天有什么会议？
机器人：[显示会议列表]

用户：把第一个改到下午3点
机器人：[理解"第一个"指的是刚才列表中的第一个会议]
```

### 3. 自然表达

不需要使用特定格式，用自然语言即可：
```
✅ 明天下午有什么安排？
✅ 帮我找一下张三
✅ 创建一个会议纪要
✅ 我有哪些待审批？
```

## 🎯 最佳实践

1. **明确表达意图**
   - ✅ "查看明天的日程"
   - ❌ "明天"

2. **提供必要信息**
   - ✅ "创建明天下午3点的会议"
   - ❌ "创建会议"（缺少时间）

3. **确认前仔细检查**
   - 重要操作会显示详细信息
   - 确认前请仔细核对

4. **善用查询功能**
   - 先查询再操作
   - 避免重复创建
